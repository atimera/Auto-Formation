✔️ Réponse correcte (niveau solide)

👉 On ne peut pas multiplier deux matrices n’importe comment car :

il faut que :
nombre de colonnes de A = nombre de lignes de B
🔁 Pourquoi ?

Parce que :

👉 chaque calcul =
produit scalaire ligne × colonne

❌ Exemple impossible :
(2,3) × (2,2)

👉 impossible car :

3 ≠ 2

👉 on ne peut pas aligner les valeurs

✔️ Intuition simple

👉 Tu fais correspondre :

les features
avec les poids
🧠 Version ML
X = données (n, m)
W = poids (m, 1)

👉 m doit correspondre sinon :

modèle incohérent
